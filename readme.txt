pbfirstkill plugin (v1.6) for B3

Requirements
------------

* BigBortherBot(3) >= version 1.10

Installation
------------

1. Copy the 'pbfirstkill' folder into 'b3/extplugins' and 'pbfirstkill.ini' file into '/b3/extplugins/conf'.

2. Open your B3.ini or b3.xml file (default in b3/conf) and add the next line in the [plugins] section of the file:
    for b3.xml
        <plugin name="pbfirstkill" config="@b3/extplugins/conf/pbfirstkill.ini"/>
    for b3.ini
        pbfirstkill: @b3/extplugins/conf/pbfirstkill.ini
